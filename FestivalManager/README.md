# Festival Management System 
 
## خلاصه پروژه 
این سامانه .برای مدیریت و هماهنگی جشنواره‌ها و رویدادهای فرهنگی شهر طراحی شده است. هدف، ساختاردهی اطلاعات رویدادها و شرکت‌کنندگان با رعایت اصول توسعه مدرن است. 
 
استانداردهای اصلی: 
*   توسعه: Pair Programming و Group Coding Workflow 
*   کنترل نسخه: Git 
*   کدنویسی: STL Containers و Exception Handling 
 
--- 
 
## ویژگی‌های کلیدی 
1.  مدیریت موجودیت‌ها: کلاس‌های رویدادها و شرکت کنندگان با ویژگی‌های تعریف شده. 
2.  ثبت‌نام: متد registerParticipantToEvent با مدیریت خطا (Exception) برای ظرفیت تکمیل شده یا شرکت‌کننده نامعتبر. 
3.  جستجو و لیست‌بندی: امکان یافتن بر اساس ID و نمایش لیست رویدادها بر اساس نوع یا تعداد شرکت‌کننده. 
 
--- 
 
##  ساختار فایل‌ها و توسعه 
فایل‌ها بر اساس ساختار پیشنهادی (main.cpp, Event.*, Participant.*, Manager.*) سازماندهی شده‌اند. 
 
نکات فنی توسعه: 
*   استفاده از std::map برای نگهداری داده‌ها. 
*   استفاده از Header Guards در فایل‌های هدر. 
*   کامیت‌های منظم با پیام‌های واضح انگلیسی.
